Capture reliability checklist status and export a JSON runbook report.

== Description ==

Kinsta Reliability Audit provides an admin interface to track hosting reliability checklist items and export structured JSON runbook reports for operational use.

== Installation ==

1. Upload the `kinsta-reliability-audit` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the Reliability Audit admin page.

== Frequently Asked Questions ==

= Does this require a Kinsta hosting account? =

No, the checklist is applicable to any WordPress hosting environment.
